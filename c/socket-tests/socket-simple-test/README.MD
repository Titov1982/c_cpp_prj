# Самый простой пример работы с сокетами

Сервер:
    1.Открывает порт на прослушивание (по умолчанию = 12345)
    2.Принимает соединение от клиента
    3.Читает данные, которые послал ему клиент
    4.Печатает эту строку в консоль
    5.Отправляет эхо на сторону клиента

Клиент:
    1. Подключается к серверу (ip: localhost, port: 12345)
    2. Посылает на сервер строку
    3. Читает ответ сервера и выводит его в консоль
    4. Закрывает соединение

В качестве клиента можно использовать telnet:

    telnet 127.0.0.1 23456

Проверка открытых дескрипторов и портов процесса:

    lsof -p pid
