# Простой генератор псевдослучайных чисел основанный на функции стандартной библиотеки random().
### Функции генератора вынесены в отдельнцю статическую библиотеку.

Программа принимает два параметра, первый из них принимает название библиотеки, а второй кол-во значений, которое необходимо сгенерировать.

name = glibc
params = 10

Пример вызова программы: ./main glibc 10
